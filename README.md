# Checklists by Bejhan - Final MVP

## Installation

1. In Supabase SQL Editor ausführen:
   `sql/final_policies_patch.sql`

2. Diese Dateien ins GitHub Repository hochladen/ersetzen:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon.svg`

3. GitHub Pages öffnen mit:
   `https://bejhanpacadzioski.github.io/Checkliste/?v=final-working`

## Funktionen

- Login mit Supabase
- Super Admin Erkennung
- Vorlagen erstellen
- Standardvorlagen ergänzen
- Checklisten aus Vorlagen erstellen
- Kategorien und Einträge kopieren
- Checklisten abhaken
- Fortschritt anzeigen
- Einträge hinzufügen
- Checklisten duplizieren
- Checklisten archivieren
- Benutzerverwaltung für bestehende Supabase Auth User
- Admin kann als anderer User arbeiten
